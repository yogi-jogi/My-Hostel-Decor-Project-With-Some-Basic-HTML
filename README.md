# My Hostel Decor Project With Some Basic HTML

This is a basic HTML project that demonstrates a hostel room decoration website. It focuses on fundamental web development concepts like page structure, layout design, and image integration. Ideal for beginners learning frontend development.

---

### 🚀 What I Mastered in this Project:

* **Semantic HTML5 Layout:** Structured the page properly using `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, and `<footer>` tags.
* **Advanced Tables & Lists:** Organized budget expenses using a data table with integrated captions, headers, and column merging (`colspan`). Also utilized various ordered (`<ol>`) and unordered (`<ul>`) list types.
* **Interactive Forms:** Built an advanced "Contact & Feedback" form utilizing diverse inputs like text fields, email, radio buttons, checkboxes, range sliders, date, and time selectors.
* **Media & Navigation:** Embedded a responsive YouTube video using an `<iframe>` for layout inspiration and set up smooth anchor tags for section navigation.
* **Modern Web Standards:** Ensured zero errors and warnings by strictly validating the markup against official W3C standards (removing obsolete tags like `<big>`).
